Selenium autotests for GitHub with use Allure

### Dependencies:

* Java 8
* Selenide 5.16.1
* JUnit 5.7.0
* Aspectj 1.9.4
* Allure-selenide 2.13.7
* Allure plugin 2.8.1

For run tests need add credentials.properties with fields in resources
> login_github=
> 
> password_github=

Then run tests with use terminal:
> gradle test
